# ✨ Modern Form Validation UI

A responsive contact form with real-time JavaScript validation and a modern glassmorphism design.

---

## 🚀 Features
- ⚡ Real-time validation
- 👤 Full name check (first + last name)
- 📞 Phone number validation (10 digits only)
- 📧 Email format validation
- 💬 Message length validation (min 30 characters)
- 📱 Fully responsive design
- 🎨 Glassmorphism UI
- ✨ Smooth hover & focus effects

---

## 🛠️ Tech Stack
- 🌐 HTML5
- 🎨 CSS3
- ⚙️ JavaScript (Vanilla)

---

## 📏 Validation Rules
- 👤 Name: Must include first + last name  
- 📞 Phone: Numbers only, exactly 10 digits  
- 📧 Email: Must follow valid email format  
- 💬 Message: Minimum 30 characters required
